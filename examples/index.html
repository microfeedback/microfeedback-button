Wishes Button Demo

wishes-button is a small (< 3 kb gzipped) library for adding a feedback button to your site.

Use this library together with a wishes microservice, such as micro-wishes-github, which will record user feedback as GitHub issues.

Basic usage

To create the default button:

wishes({ url: ... });

Binding to a specific element

To bind to a specific element:

wishes(document.getElementById('feedback-button'), { url: ... });
