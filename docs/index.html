microfeedback-button

microfeedback-button is a small library for adding a feedback button to your site.

It uses sweetalert2 for displaying beautiful, responsive input dialogs.

Use this library together with a microfeedback backend, such as microfeedback-github, which will record user feedback as GitHub issues.

Default button

The default button with no configuration is at the bottom right of the page.

Binding to a specific element

You can bind the feedback dialog to any element.

Success message after response

By default, the "thank you" message will be shown as soon as the user submits their input. Set optimistic: false if you want to show the message only after the request to the backend is complete.

Handling failure

Use onFailure() to handle failed requests when optimistic is false.

Custom success dialog

If optimistic is set to true, use the preSend() hook to customize the success dialog.

If optimistic is set to false, use the onSuccess() hook to customize the success dialog.
