microfeedback-button

microfeedback-button is a small (~3 kb gzipped) library for adding a feedback button to your site.

Use this library together with a microfeedback backend, such as microfeedback-github, which will record user feedback as GitHub issues.

Basic usage

To create the default button:

microfeedback({ url: 'https://your-microfeedback-backend.now.sh' });

Binding to a specific element

To bind to a specific element:

microfeedback(document.getElementById('feedback-button'), { url: ... });

Submit callback

To create the default button:

microfeedback({
  onSubmit: function(e, value) {
    alert('Feedback submitted: ' + value);
  },
});

Screenshots (beta)

Enable screenshots:

microfeedback(document.getElementById('feedback-button-screenshot'), {
  url: ...,
  // Capture everything within div#container
  screenshot: document.getElementById('container'),
});
